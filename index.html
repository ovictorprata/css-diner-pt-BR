Você está prestes a aprender os Seletores CSS! Seletores são como você escolhe qual elemento aplicar estilos.

Exibição 1 - Uma Regra CSS

      p {
         margin-bottom: 12px;
      }

Aqui, o "p" é o seletor (seleciona todos os elementos <p>) e aplica o estilo margin-bottom.

Para jogar, digite um seletor CSS no editor abaixo para selecionar os itens corretos na tabela. Se você acertar, avançará para o próximo nível.

Passe o mouse sobre os itens na tabela para ver o markup HTML deles.
